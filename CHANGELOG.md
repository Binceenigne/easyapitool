# 更新日志

## 1.0.13 - 2026-07-23

- 新增窗口置顶功能，方便在查看或操作其他应用时持续关注用量。
- 默认和极简标题栏可从左侧图钉按钮切换置顶；Windows 原生标题栏可从设置按钮右侧切换。
- 置顶状态会自动保存，进入低开销模式后重新打开窗口仍然有效。
- 修复 GitHub 接口访问次数用尽时无法检查更新的问题，应用会自动切换到备用检查方式。

## 1.0.12 - 2026-07-23

- 检查更新时只显示尚未安装版本的更新内容，不再一次展示全部历史记录。
- 更新窗口新增“查看完整更新日志”，需要时可以展开查看完整更新路线。
- 优化更新说明，用更直白的语言说明每项变化及其影响。

## 1.0.11 - 2026-07-23

- 应用行为新增三种后台运行方式，可按内存占用和打开速度自由选择。
- 默认在隐藏 5 分钟后进入低开销模式，再次打开窗口时会重新加载界面。
- 低开销模式可降低约 80% 的后台内存和运行开销，但再次打开窗口会稍慢。
- “始终保持活跃”会保留界面，重新打开速度更快，但后台内存占用较高。
- 无论选择哪种模式，后台都会继续自动获取用量并保存历史数据。

## 1.0.10 - 2026-07-22

- 修复了点击刷新按钮有概率失效的问题。
- 修复了添加、删除、设置等其他按钮偶尔点击无响应的问题。
- 优化后台运行逻辑，隐藏窗口后可以释放界面占用的内存。
- 修复设置页面被顶部按钮遮挡的问题。

## 1.0.9 - 2026-07-22

- 修复部分网络环境下无法检查更新的问题。
- 刷新按钮点击后立即进入 5 秒冷却，避免重复刷新。
- 刷新按钮新增加载、成功、失败和冷却倒计时反馈。
- 增加刷新过程日志，方便排查刷新失败或按钮无响应的问题。

## 1.0.8 - 2026-07-20

- 修复更新完成后应用偶尔无法自动重新启动的问题。
- 修复最小化到托盘时应用偶尔无响应的问题。
- 修复部分标题栏按钮偶尔点击无响应和窗口恢复时闪烁的问题。
- 优化密钥选择菜单，支持键盘操作并统一不同标题栏下的体验。
- 调整极简标题栏的尺寸和按钮大小，操作更方便。

## 1.0.7 - 2026-07-17

- 修复应用重新启动后偶尔出现页面文件丢失的问题。
- 优化极简标题栏的高度、图标和窗口按钮大小。
- 极简模式支持更小的窗口尺寸。

## 1.0.6 - 2026-07-17

- 新增默认、极简和 Windows 原生三种标题栏样式。
- 极简标题栏会收起常用操作，为内容留出更多空间。
- Windows 原生标题栏支持系统自带的拖动、缩放和窗口按钮。

## 1.0.5 - 2026-07-16

- 优化更新下载进度显示。
- 修复下载完成后偶尔无法继续安装的问题。
- 更新完成后可以选择稍后重启或立即重启。
- 修复无限额度账户可能显示错误额度上限的问题。

## 1.0.4 - 2026-07-16

- 新增自动检查更新和应用内更新提示。
- 支持暂时跳过或忽略指定版本。

- 修复正常用量变化可能被误报为额度上限调整的问题。
- 更新应用图标和 Windows 通知图标。

## 1.0.3 - 2026-07-16

- 额度上限或频率限制变化时会发送 Windows 通知。
