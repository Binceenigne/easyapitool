# 更新日志

## 1.0.2 - 2026-07-16

- 默认余量报警阈值调整为 50%、25% 和 10%。
- 更新进度条改为 4×4 点阵随机填充，空闲时保持熄灭。
- 更新日志改用更清晰的大字号中文 Markdown 排版。
- Windows 通知按正常、一级、二级和严重状态使用高清点阵机能风图标。

## 1.0.1 - 2026-07-15

- 修复频率窗口已用额度略高于上限时，可用额度被重复计算为负数的问题。
- 频率窗口现在优先采用上游返回的剩余额度，并统一钳制在 `0` 到窗口上限之间。
- 超额时仍保留实际已用额度展示，但剩余比例最低为 0%、已用比例最高为 100%。

## 1.0.0 - 2026-07-15

- 新增 GitHub Release 自动更新，下载进度使用自适应点阵进度条。
- 新增每次启动、每周或手动检查更新频率设置。
- 新增关闭应用、最小化到托盘或每次询问的关闭按钮行为。
- 新增 Windows 开机自启设置。
- 新增应用内更新日志展示。
- 修复点阵进度条在不同窗口宽度下越过父组件的问题。
- 优化点阵渲染与关键动态节点的 GPU 合成，减少重复 DOM 重建。
- 为内部模态框增加平滑进退场动画，更新下载点阵使用 6 秒长缓动连续展示进度。
